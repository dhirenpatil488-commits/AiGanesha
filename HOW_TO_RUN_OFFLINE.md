# Offline Execution Guide

Since this is a React Single Page Application (SPA) without a complex backend, I have compiled a standalone production version for you. You do not need to install any servers or complex software to run it.

1. Navigate to: `D:\GHG Calculator\`
2. I have packaged the offline application into this zip archive: `GHG_Calculator_App.zip`
3. Extract the contents of this `.zip` file into a folder of your choice on your desktop.
4. Inside that extracted folder, simply double-click the `index.html` file to open it in your default web browser.

Because we updated the Vite configuration to use relative paths (`base: './'`), all CSS and JavaScript assets will load perfectly right from your local filesystem. You can use it anywhere, anytime!
